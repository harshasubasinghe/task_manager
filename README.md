# task_manager

//Requirements
<ul>
<li>Laravel</li>
<li>PHP 7</li>
<li>Composer</li>
<li>MySQL</li>
</ul>

A webbased systems taht lets you,

<ul>
<li>Create tasks</li>
<li>Assign tasks</li>
<li>Remove tasks</li>
</ul>

Tasks can be created to have following attributes

<ul>
<li>Start time</li>
<li>End time OR Duration</li>
<li>Priority</li>
<li>Completion reward</li>
</ul>
